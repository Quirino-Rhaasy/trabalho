# Sistema de Gerenciamento de Tarefas

Projeto desenvolvido para a disciplina de Engenharia de Software.

## Objetivo

Criar um sistema CRUD para gerenciamento de tarefas utilizando metodologia ágil.

## Tecnologias

- Python
- Flask
- Pytest
- GitHub Actions

## Funcionalidades

- Criar tarefa
- Listar tarefas
- Atualizar tarefa
- Excluir tarefa

## Metodologia

Foi utilizada a metodologia Kanban para organização das atividades.

## Mudança de Escopo

Inicialmente o sistema apenas cadastrava tarefas.

Durante o desenvolvimento foi adicionada a funcionalidade de marcar tarefas como concluídas para melhorar o controle das atividades.
